Social Wishlist
Web application where users create wishlists and share them with friends.

Friends can reserve gifts so that no one buys the same present twice. If the gift is expensive, multiple friends can contribute.

The wishlist owner cannot see who reserved the gift to keep the surprise.

Features
• User registration and login
• Create wishlists
• Add gifts (title, link, price, image)
• Automatic product preview from URL
• Public wishlist link
• Gift reservation system
• Real-time updates

Tech Stack
Next.js
React
Supabase
PostgreSQL
TailwindCSS

Run locally
npm install npm run dev

Open: http://localhost:3000
